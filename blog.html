Important FAQ

What is the Document Object Model(DOM)?

The DOM is a programming interface that represents the structure of an HTML or XML document as a tree of objects. Each node is an object representing parts of the document (elements, attributes, text), enabling dynamic interaction with the webpage.

How do you select an element from the DOM?

We can select elements using methods like document.getElementById('id'), document.querySelector('.class or #id'), or document.getElementsByClassName('class'). These methods return references to elements for manipulation.

What is Event Delegation in the context of the DOM, and why is it useFul?

Event delegation involves attaching a single event listener to a parent element instead of multiple child elements. It leverages event bubbling, making it more efficient for handling dynamic elements and reducing memory usage.

How do you manipulate an element's attribute and styles using the DOM?

We can use methods like element.setAttribute('attribute', value) to modify attributes. To change styles, use element.style.property = 'value' or manipulate the class list using classList.add(), remove(), or toggle().
